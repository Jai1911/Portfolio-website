Travel rate comparison platform

This is a platform developed to ease the process of viewing all the travel rates while choosing the ideal mode of transport. This platform first asks the user for the destination and then displays the prices for all available travel modes .ie. Flights, Trains, Bus, Car, all at the same time. This helps the user to choose more efficiently.

Currently this platform has 2 versions: Version 1: There is a pre-defined databse with 30 major cities and a defined origin (Jaipur), this was developed to show the working of the platform and does not show real-time prices.

Version 2: This version does not have any pre-defined database and web-scraping tools are used to fetch the real-time data. However web-scraping is not ethical and we are in the process of searching for a work-around by using API keys if they are available. The front-end work for creating a GUI is currently in progress.
